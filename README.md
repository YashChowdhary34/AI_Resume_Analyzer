# AI Resume Analyzer

## Overview
AI Resume Analyzer is a web-based application that analyzes resumes and provides intelligent insights to improve candidate profiles. It leverages Natural Language Processing (NLP) to extract key information and offers recommendations based on skills, experience, and job alignment.

## Features
- Resume parsing using NLP techniques
- Skill extraction and categorization
- Personalized recommendations for improvement
- Job role matching based on resume content
- User-friendly interface for uploading and analyzing resumes
- Basic analytics and visualization of user data

## Tech Stack
- Python
- Streamlit (Frontend)
- NLP libraries (spaCy / NLTK)
- Machine Learning concepts
- CSV / Database handling

## Installation

1. Clone the repository
```
git clone <your-repo-link>
cd <repo-folder>
```

2. Install dependencies
```
pip install -r App/requirements.txt
```

3. Run the application
```
streamlit run App/App.py
```

## Usage
- Upload a resume (PDF format recommended)
- The system will extract relevant details
- View insights such as:
  - Skills detected
  - Suggested improvements
  - Recommended learning paths
